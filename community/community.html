Important

Please read the Code of Conduct before participating in xw3 community discussions.

IRC

We use the Libera.Chat IRC network for realtime collaboration.

Just click the channel name to open a web based client in your browser for instant chat.

Channel Description
#xw3 Offtopic discussion about everything
#xw3-dev Developer discussion about xw3
#xw3-usr User discussion about xw3

Mailinglists

Mailinglists are provided by the lists.xw3.org mailinglist server.

Simply click "Subscribe" to join a list by sending a blank email.
List Description Action
xw3@lists.xw3.org Offtopic discussion about everything [Subscribe] [Unsubscribe]
xw3-announce@lists.xw3.org For xw3 announcements [Subscribe] [Unsubscribe]
xw3-developers@lists.xw3.org Developer discussion about xw3 [Subscribe] [Unsubscribe]
xw3-users@lists.xw3.org User discussion about xw3 [Subscribe] [Unsubscribe]

You can also post to a list without a subscription, but your messages will be moderated and experience indefinite delays in delivery, and you won't receive replies if other users simply reply to the list without CCing you.

Mastodon

We will use Mastodon for announcements in the future: https://chaos.social/@xw3.

Others

We believe that the communication channels mentioned above are sufficient for xw3 development. IRC, mailing lists and Mastodon provide you with the perfect entry points for discussions about xw3 development. Nothing more is needed.

Matrix

We do not and will never provide chat over the Matrix protocol in any way. Matrix sucks!

We also will in no way provide an IRC <-> Matrix bridge. Again, Matrix sucks!
