About

xw3.org is mostly a template for Jekyll based websites PLUS a cgi directory for adding dynamic features like a contact form.

You can use the Jekyll template even when not using the cgi features or you can use the cgi even when not using the Jekyll template. All you require is the JavaScript library jQuery when using the cgi. When only using the template jQuery is not required.

xw3.org is no software, it's an idea... Like a proof-of-concept.
